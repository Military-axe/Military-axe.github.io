Dll hide in PE

2021-06-03

Intro: hide dll in PE. When I run PE. Mapping it to memory then analyze it and call the function in dll. Next time I want to hide some shell in dll.

STEP

  • pushing dll in memory as a resource fill
  • getting dll’s image size by dll’s pe struct
  • malloc a place(rwe) by size
  • formatting dll data as SectionAlignment and mapping it to malloc memory
  • fixing the Relocation Table data
  • loading dll and getting function address by Import Address Table
  • updating dll load address to step 3 memory address

RECORD

First I build a dll file as x64 release mode. Just a pop-up windows show dll载入成功

Then create a project by vs2019. Then create a class to write a function.
