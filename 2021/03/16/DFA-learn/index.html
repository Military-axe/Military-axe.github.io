DFA learn

2021-03-16 CRYPTO

Intro: DES one round DFA attack principle

DFA attack should use together with CPA(Choose plaintext attack)

I with use DES sbox1 as example.

We know DES sbox characteristics is 6bit input -> 4bit output.It mean that 64 possibilities will be 16 possibilities. This feature can help we to attack.

Parameters

Parameters means
PPP First plaintext
P∗P^*P∗ Another plaintext
P′P'P′ P⊗P∗P\otimes P^*P⊗P∗
III sBox input
I∗I^*I∗ Another sbox input
I′I'I′ I⊗I′I\otimes I'I⊗I′
OOO output
O∗O^*O∗ Another output
O′O'O′ O⊗O′O\otimes O'O⊗O′
KKK key

Encryption

In this scenario. We can reverse it simply. But this just nor round, how about 2 round?

How to attack

If I can get III so I can calculate $k = I\otimes $P.

P′=P⊗P∗=(I⊗k)⊗(I∗⊗k)=I⊗I∗=I′P'=P\otimes P^*=(I\otimes k)\otimes(I^*\otimes k)=I\otimes I^*=I'P′=P⊗P∗=(I⊗k)⊗(I∗⊗k)=I⊗I∗=I′

So I search all xor input and xor output as a DFA table

I can find III by search table. Then I can get kkk

DFA table

Exhaustive enumeration all possibilities to get DFA table.

Attack

If I input P=1,P∗=0x35P=1,P^*=0x35P=1,P∗=0x35. So P′=P⊗P∗=0x34P'=P\otimes P^*=0x34P′=P⊗P∗=0x34

So I searched DFA table to know It have 8 possibilities.

Add some a DFA table file to computer that values

So we get 8 III value 0x06、0x10、0x16、0x1c、0x22、0x24、0x28、0x32

Use P,P∗P,P^*P,P∗ xor with all I can get all possible key.

The try to change P,P∗P,P^*P,P∗ then get new kkk dict. Find all key dict intersection to get the real key.

Attack one round and two round code

we can find the 17 in result.
